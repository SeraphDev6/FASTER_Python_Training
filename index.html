What is Python?

  • Python is a dynamic, strongly typed, general purpose programming language
  • It prioritizes readability, simplicity and quick iteration
  • As one of the most popular languages in the world it has a very large user base, and mature libraries handling most tasks

Why Use Python for data science?

  • It's readable and has a low barrier to entry
  • Python code can be integrated with a variety of applications
  • Python's data libraries are compiled in C, making them operate much more efficiently
  • Python has wide support for data analytics and machine learning
  • Jupyter Notebooks enable rapid iteration and quick discovery when analyzing data

Tools for Data Analysis in Python

  • Pandas
    • The implementation of DataFrames, Sequences, and utility function for them in Python is handled by Pandas
  • Matplotlib
    • Matplotlib handles most charting in Python but can be extended using Seaborn or Plotly

Downloading Packages

Packages can be downloaded using either pip or conda

For the purposes of this training, we will be using conda. Open a terminal and run the following command

conda create -n faster python=3.9 && conda activate faster && conda install pandas matplotlib seaborn
